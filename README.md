# 🍔 Food Ordering GUI Application

![Python](https://img.shields.io/badge/Python-3.8+-blue?logo=python)
![MySQL](https://img.shields.io/badge/MySQL-8.x-orange?logo=mysql)
![Tkinter](https://img.shields.io/badge/GUI-Tkinter-lightblue)
![License](https://img.shields.io/badge/License-MIT-yellow)

A **desktop GUI application** for food ordering built with Python. Features a graphical interface for browsing menu items, placing orders, and storing transaction data in a MySQL database.

---

## 📌 What This Project Does

- Provides a **graphical food ordering interface** using Python Tkinter
- Stores and retrieves order data using a **MySQL database**
- Allows users to browse menu items, select quantities, and confirm orders
- Demonstrates full-stack desktop app development with Python

---

## 🧰 Tech Stack

| Tool | Purpose |
|---|---|
| `Python 3.8+` | Core application logic |
| `Tkinter` | GUI framework (desktop interface) |
| `MySQL 8.x` | Order data storage |
| `mysql-connector-python` | Python ↔ MySQL connection |
